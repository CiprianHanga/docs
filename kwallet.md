## Kwallet

- config file: ```~/.config/kwalletrc```
- password files dir: ```~/usr/share/kwalletd/```

### Disable

- set ```Enabled: false``` in kwalletrc

### Enable (back)

- set ```Enabled: true``` in kwalletrc

### Remove all saved info and start pristine

- remove ```~/usr/share/kwalletd```
